<H2>Pseudoklasy (*Pseudo-classes*)</H2>
<Content>
Pseudoklasa jest specjalnym słowem kluczowym dodanym już w pierwszej specyfikacji CSS , która ma na celu pobieranie elementów w specyficznym stanie. Dla przykładu pseudoklasa **:hover** pozwala nam zdefniować style dla przycisku - ***`<button>`*** - w momencie najechania na niego kursorem przez użytkownika. 
</Content>
<Content>
</Content>
<Code>
        button:hover{
            background:#f1f1f6;
            color:#000;
            transition:.12s ease-in-out background, 
            .12s ease-in-out color;
        }
</Code>
<Content>
Pseudoklasy umożliwiają również w intuicycjnym poruszaniu się po serwisie, np. po przez podpowiedzeniu userowi (*użytkownikowi*) jakie strony już zdołał odwiedzić implementując do ```<a></a>``` w CSS pseudoklasę ***:visited***.
</Content>

<H2>Spis wszystkich pseudoklas</H2>
<Content>
</Content>
<Content>
    - **:active** aktywne linki
    - **:autofill** autouzupełnione pola formularza
    - **:checked** zaznaczony element formularza ```<input type='checkbox'/>```
    - **:disabled** nieaktywny przycisk, link bądź formularz
    - **:empty** elementy [HTML](/blog/czym-jest-html-historia-czy-terazniejszosc) nie posiadające żadnej treści w sobie ```<div></div>```
    - **:enabled** wizualna prezentacja elementów formularza, które możemy edytować lub uzupełniać
    - **:first-child** pierwsze dziecko danego typu, np. ```div:first-child{backgroud-color:pink}```
    - **:first-of-type** pierwszy typ określonego elementu
    - **:focus** prezentacja elementu, na którym ostatnio była skoncentrowana uwaga
    - **:hover** animacja najechania kursorem na element HTMl
    - **:invalid** błędnie uzupełnione pole formularza
    - **:lang** określenie styli dla poszczególnych języków, jest to przydatne dla stron multijęzykowych obsługujących np. alfabet łaciński oraz cyrylicę, istotne zwłaszcza w sytuacji kiedy w Polsce jest sporo [uchodźców z Ukrainy w wyniku wojny](https://www.facebook.com/permalink.php?story_fbid=124277603497520&id=103383665586914), którzy posługują się właśnie wyżej wymienioną cyrylicą
    - **:last-child** ostatnie dziecko wybranego przez nas elementu
    - **:last-of-type** ostatni typ wybranego przez nas elementu
    - **:link** nieodwiedzone linki
    - **:not()** wskazanie elementów które nie mają być ostylowane ```div:not(p){background-color:lime}```
    - **:nth-child()** wskazanie dowolnego w kolejności dziecka określonego elementu np. ```div:nth-of-child(2){color:red}``` ustawi dla drugiego ```<div>RadWEB</div><div>Strony internetowe i SEO</div>``` kolor czcionki na czerwony
    - **:nth-of-type()** dobranie styli dla konkretnego elementu w *kolekcji* HTML 
    - **:required** nadanie styli elementom formularza, które wymagają uzupełnienia, takie elementy są oznaczone atrybutem ***required*** ```<input type='text' id='example' name='example' required />```
    - **:root** tzw. korzeń stylów css, gdzie można zdefiniować chociażby zmienne css
        ```
            :root{
                --red:#d01;
                --blue:#00f;
                --white:#fff;
            }
        ```
    - **:valid** nadanie styli polom formularza, które zostały poprawnie uzupełnione
    - **:visited** ostylowanie linków, które zostały odwiedzone przez użytkownika
</Content>
